# 单片机控制器

这是一个基于Web的单片机控制器，使用Node.js和WebSocket实现实时通信。

## 功能特点

- LED控制（开关）
- 电机控制（正转、反转、停止）
- PWM值调节（0-255）
- 传感器数据显示（温度、湿度、光照）
- 实时通信

## 所需工具

1. **Node.js** (v14.0+)
2. **npm** (Node.js包管理器)
3. **串口驱动**（根据单片机型号安装）

## 安装步骤

1. 克隆或下载本项目到本地

2. 安装依赖包：
   ```bash
   npm install
   ```

3. 修改串口配置：
   - 打开 `server.js` 文件
   - 修改第21行的串口名称为实际使用的串口（例如 COM3、/dev/ttyUSB0 等）

4. 启动服务器：
   ```bash
   # 生产模式
   npm start
   
   # 开发模式（自动重启）
   npm run dev
   ```

5. 打开浏览器访问：
   ```
   http://localhost:3000
   ```

## 与单片机通信协议

### 从服务器到单片机

- LED控制：`LED:on` 或 `LED:off`
- 电机控制：`MOTOR:forward`、`MOTOR:backward` 或 `MOTOR:stop`
- PWM控制：`PWM:128`（值范围0-255）

### 从单片机到服务器

传感器数据格式：`TEMP:25.5,HUM:65.2,LIGHT:500`

## 注意事项

1. 确保单片机已正确连接到电脑
2. 确保串口驱动已安装
3. 修改 `server.js` 中的串口名称为实际使用的串口
4. 单片机端需要实现相应的串口通信代码
